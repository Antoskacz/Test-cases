
📘 Návod na použití aplikace pro generování test case

1. Ujisti se, že máš nainstalovaný Python 3.
2. Otevři konzoli v této složce.
3. Spusť aplikaci příkazem:
   python main.py
4. V menu zadej věty nebo uprav kroky.
5. Výsledný soubor 'vystup.xlsx' bude vytvořen v této složce.

Soubor 'kroky.json' obsahuje šablony kroků, které se automaticky párují podle klíčového slova ve větě.
